# Galactic Spacefarer Manager
The Galactic Federation's official Galactic Spacefarer Adventure management application

# Setup
Follow [capire's Initial Setup section](https://cap.cloud.sap/docs/get-started/#setup) for first steps.

```
git clone https://github.com/Laczi97/galactic-spacefarer-manager.git
cd galactic-spacefarer-manager
```

```
npm install
```

```
cds deploy
```

# Run
```
cds watch
```

http://localhost:4004/

http://localhost:4004/com.spacefederation.spacefarers.spacefarermanager/index.html

Mock authorization is found in package.json
